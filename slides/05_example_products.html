Example products

1 · Clod Bathos, Superior Machine

An LLM-driven IDN, dialogue-based. Three language models run locally, for TTS (works well), STT (not so consistent), and rewording and state-machine navigation (started working after I had Fable 5 write a testing harness and use it to iteratively debug the local small language model so I didn't have to playtest through the playtime interface). The main difficulty in this approach is balancing stupidity of small language models versus resource requirements of large ones.
