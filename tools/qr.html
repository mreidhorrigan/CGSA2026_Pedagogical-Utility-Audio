QR code for your presentation

Turn the address where you’re serving the game into a scannable code to put on a slide. Everything here runs offline — nothing is uploaded.

Put it on a slide two ways: drop the downloaded join.html into slides/ and run python3 build_slides.py — or click Copy slide HTML and paste it into slides.js as a { type:"html", html:"…" } entry.
Tip: open this page as tools/qr.html?url=YOUR_SERVING_URL to pre-fill the box automatically (the multiplayer server serve.py prints a ready-made link).
